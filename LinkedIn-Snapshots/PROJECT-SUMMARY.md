## 🔧 Customization Guide

### Change School Name
Find in each HTML:
```html
<div class="school-name">🏫 PREMIER INTERNATIONAL SCHOOL</div>
```
Replace with your name

### Update Numbers
Search for these metrics and update:
```
1,245 → Your student count
87 → Your staff count
28 → Your class count
₹12.5L → Your fee data
```

### Modify Colors
Edit the CSS `<style>` section:
```css
primary: #009FBC → Your primary color
secondary: #E7746F → Your secondary color
```
